# EU Industrial KPI Analysis
End-to-end analysis of EU manufacturing performance indicators (2018–2023), including KPI computation, interactive dashboards, and structured data workflow.
 
## Dataset
All data used for this analysis is stored in the repository under the `data/` folder:
- **Processed Data**: `processed/nama_10_a64_manufacturing_2018_2023_pulita.xlsx`

## Tools & Technologies
- Excel / Power Query / Power Pivot
- Power BI
- Version Control (Git/GitHub)
- Structured folder organization
- Methodological rigor

## Methodology
1. Data Acquisition: Eurostat National Accounts (2018–2023)
2. Data Cleaning: missing values, standardization, harmonization
3. KPI Computation: value added, growth rates, sector metrics
4. Data Organization: processed datasets in `data/processed/`
5. Visualization & Reporting: Power BI dashboards for trends and insights
6. Reproducibility: documented workflow for replication

## Key Insights
- EU manufacturing value added increased steadily from 2018 to 2023, with acceleration in 2021
- Some countries consistently outperform in growth and productivity
- KPI trends indicate increasing efficiency in selected manufacturing sectors
